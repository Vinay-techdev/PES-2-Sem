# Analysis and Design of Algorithms (AAD)

This repository contains my hands-on implementations and practice code for the **Analysis and Design of Algorithms (AAD)** course during my MCA at **PES University**.

## 📚 Topics Covered

* Algorithm Analysis (Time & Space Complexity, Asymptotic Analysis)
* Sorting Algorithms (Bubble, Selection, Insertion, Merge, Quick, Intro Sort)
* Searching Algorithms (Linear, Binary, Interpolation, A*)
* Graph Algorithms (Prim's, Kruskal's, Dijkstra's, Floyd's, Bellman-Ford, Ford-Fulkerson)
* String Matching Algorithms (Naive, KMP, Rabin-Karp, Horspool, Boyer-Moore)
* Data Compression (Huffman Coding, LZ77, Run-Length Encoding)
* Complexity Theory (Decision Trees, P, NP, NP-Complete)
* Combinatorial Optimization (0/1 Knapsack, N-Queens, TSP, Subset Sum, Job Scheduling, Hamiltonian Circuit)

## 💻 Repository Overview

This repository includes implementations of **selected algorithms and concepts** that I practiced as part of coursework and self-learning.

Some topics are represented with complete code implementations, while others were studied conceptually or demonstrated during laboratory sessions.

## 🚀 Future Plans

I plan to continue expanding this repository by implementing additional algorithms and optimization techniques that were covered in the syllabus but are not yet included.

## 🎯 Goal

This repository serves as my learning journal for algorithm design and problem solving, and I will keep updating it as I explore more advanced concepts.
