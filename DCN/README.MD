# Computer Networks (CN)

This repository contains my hands-on implementations, network simulations, and lab exercises for the **Computer Networks (CN)** course during my MCA at **PES University**.

## 📚 Topics Covered

* Internet Architecture & TCP/IP Model
* Network Edge & Core
* Protocol Layers and Services
* HTTP, DNS, Electronic Mail
* TCP & UDP
* Reliable Data Transfer (Stop-and-Wait, GBN, SR)
* IPv4, IPv6, DHCP, ICMP, NAT
* Routing Algorithms (RIP, OSPF, BGP)
* Link Layer (ARP, Ethernet, VLANs, MPLS)
* SDN Fundamentals
* Error Detection (Parity, Checksum, CRC)

## 💻 Repository Overview

This repository includes **selected networking programs, Packet Tracer simulations, socket programming examples, and protocol analysis** completed during coursework and practical sessions.

Some concepts are implemented through code and network simulations, while others were explored theoretically and through Wireshark analysis.

## 🛠 Tools & Technologies

* Cisco Packet Tracer
* Wireshark
* Socket Programming (TCP & UDP)
* Java / Python (where applicable)
